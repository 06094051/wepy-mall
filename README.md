### 安装（更新） wepy 命令行工具。
	cnpm install wepy-cli -g

### 安装依赖包
	cnpm install

### 开发实时编译。
	npm run dev

### 生产
	npm run build

### 开发使用说明

使用微信开发者工具新建项目，本地开发选择dist目录。

微信开发者工具 --> 项目 --> 关闭ES6转ES5。

本地项目根目录运行npm run dev，开启实时编译。

### wepy开发文档地址
	https://wepyjs.github.io/wepy/#/

### 小程序开发文档
	http://mp.weixin.qq.com/debug/wxadoc/dev/
